# Carousel in React

This React app ([see GitHub Page](https://hollyw00d.github.io/carousel-react/)) shows a carousel using:

- A mock API to pull in image URLs and alt text
- Accessibility features, including <code>aria-live="polite"</code> to show when the carousel has been updated and read out the image slide on [VoiceOver (screen reader for Mac)](https://en.wikipedia.org/wiki/VoiceOver) and extensive testing on VoiceOver
